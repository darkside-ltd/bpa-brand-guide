## Shapes

We provide sixteen shapes that convey movement and are designed to give some flexibility and creative freedom when implementing and using them in any medium.The shapes have been designed to follow the contours of the ParalympicsGB logo and agitos.

You have the freedom to use these in any colour from this guide, giving plenty of scope for creativity. 

We prefer to avoid showing the ends of any shapes, and recommend using linear gradients in these instances to blend them into backgrounds.As always we expect WCAG compliance to take priority when using these assets with written content to ensure accessibility.
